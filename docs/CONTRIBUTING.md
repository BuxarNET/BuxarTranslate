# Contributing to BuxarTranslate

We love your input! We want to make contributing to BuxarTranslate as easy and transparent as possible.

## Development Setup

1. Fork the repository
2. Clone your fork: `git clone https://github.com/BuxarNET/BuxarTranslate.git`
3. Create a branch: `git checkout -b feature/amazing-feature`

## Building the Extension

```bash
cd scripts
chmod +x build-xpi.sh
./build-xpi.sh
